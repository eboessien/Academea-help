Program Setup Help

About Program Setup

The Program Setup page allows you to manage the different educational programs offered by your institution. Programs can be added, edited, or deactivated as needed.

Using the Add Button

The "Add New Program" button allows you to create new educational programs in the system:

Click the "Add New Program" button located at the top right of the page.

A form will appear where you can enter the program details.

After filling in the required information, click "Save" to add the program.

Add Button Preview

Using the Action Buttons

The Action column in the programs list provides buttons to manage existing programs:

Click the (edit) icon to modify program details.

Click the (delete) icon to remove a program (you will be asked to confirm).

Programs with existing students cannot be deleted but can be deactivated.

Action Buttons Preview

How to Add a New Program

Navigate to the Program Setup page from the Configuration menu.

Click the "Add New Program" button located at the top right of the page.

Enter the Program Name in the provided field.

Select "Yes" or "No" from the Active dropdown to set the program status.

Click the "Save" button to add the program or "Cancel" to discard changes.

How to Edit an Existing Program

Locate the program you wish to edit in the programs list.

Click the edit icon (pencil) in the Action column for that program.

Modify the program details as needed.

Click "Save" to apply changes or "Cancel" to discard them.

Note

Deactivating a program will not delete it from the system but will remove it from selection options in other parts of the application.

Weeks Help

About Weeks

The Weeks section allows you to manage the academic weeks for your institution, including setting if it is active or not

Using the Add Feature

To add a new week, simply type in the Week Name field and set the active status:

Click the "Week Name" entry box to type a new week name.

Select "Yes" or "No" from the Active dropdown to set the week status.

Click "Submit" to add the new week to the system.

Add Feature Preview

Using the Action Buttons

Each week entry has action buttons for management:

Click on a week entry to select it for editing.

Modify the week details as needed in the form.

Click "Update" to save changes to the selected week.

Action Feature Preview

How to Set Up a New Week

Navigate to the Weeks page from the Configuration menu.

Click the "Week Name" entry box to type a new week name.

After entering the name, click on "Active" and select a status for your week(e.g., Week 1, Active status(Yes)).

Click on "Submit" to add the New week you've created.

Click "Reset" if you don't like your entries and would like to erase them.

Even after Updating, you can still go back, click on the week and modify its data and save it.

Note

There are four buttons above this page which help you copy to clipboard,download contents of the page ,download as file and pdf respectively.

Terminal Report Help

About Terminal Report Configuration

The Terminal Report section allows you to configure the settings and templates for student terminal reports, including grading systems and comment banks.

How to Configure Terminal Reports

Navigate to the Terminal Report page from the Configuration menu.

Set up the grading system (e.g., letter grades, percentages, or descriptive grades).

Configure the report template layout and included sections.

Create or manage comment banks for teacher remarks.

Set up signature lines and other formal elements.

Click "Save" to apply your changes.

Note

Terminal report configurations can be set differently for various programs or class levels.
