Introduction to All Students

The All Students section provides a comprehensive view of every student in your institution's database. This powerful tool helps administrators manage student information efficiently.

Note: You need appropriate administrative permissions to access all student data. Contact your system administrator if you need access.

In this guide, you'll learn how to:

  • Navigate to the All Students section
  • Understand the interface layout
  • Search and filter student records
  • Customize visible columns
  • Perform actions on student records

Navigating to All Students

To access the All Students section:

1Open the Dashboard

Log in to your AcademEA account and you'll be directed to the Dashboard homepage.

2Access the Students Menu

In the sidebar navigation, locate and click on the "Students" menu to expand it.

3Select All Students

From the expanded Students menu, click on "All Students" to open the page.

Interface Overview

The All Students page displays a table with all student records. Here's what you'll see:

1Student Table

The main content area shows a table with columns for various student information:

  • Student ID
  • First Name, Middle Name, Last Name
  • Gender, House, Nationality
  • Program, Class, Residency
  • And other demographic information
2Search Functionality

At the top of the table, you'll find a search box that allows you to quickly find specific students.

3Action Buttons

Various action buttons are available for managing students, including options to print, export, and perform other operations.

Searching for Students

Finding specific students is easy with the search functionality:

1Using the Search Box

Locate the search box at the top of the student table. Type any search term like:

  • Student ID
  • First or Last name
  • Class or Program
  • Any other student information
2Review Results

The table will automatically filter to show only records matching your search term.

3Clearing Search

To return to the full list, simply delete your search term or click the clear button (X).

Tip: Use specific terms for better results. For example, search by full Student ID rather than just partial numbers.

Managing Table Columns

You can customize which columns are visible in the student table:

1Access Column Options

Look for the "Hide/Show Columns" button or option near the table.

2Select Columns to Display

A dialog will appear with checkboxes for all available columns. Check or uncheck boxes to show or hide columns.

3Apply Changes

Click "Apply" or "Save" to update the table with your column preferences.

Student Actions

The All Students page provides several actions you can perform:

Print Student List

Generate a printer-friendly version of the student list.

Copy Data

Copy student data to clipboard for use in other applications.

Export to Excel

Export the student list to Microsoft Excel format.

Export to PDF

Export the student list to PDF format.

1Select Students

First, select one or more students using the checkboxes in the first column of the table.

2Choose Action

Click on the action button you want to perform (Print, Copy, Export, etc.).

3Complete Action

Follow any additional prompts to complete your selected action.

Troubleshooting

Here are solutions to common issues you might encounter:

Clear any search terms and check if filters are applied. Also verify your user permissions allow viewing all students.

Ensure you have selected at least one student for actions that require selection. Check your browser permissions if exporting files.

Clear your browser cache or check if you have cookies disabled. Column preferences are often stored in browser local storage.
