CONFIGURATION (SYSTEM SETUP)

To setup the system up for use, some system values have to provided by the user. To setup the system click on the Configuration menu and select the item you want to configure.

CLASS/FORM SETUP

To add a class click on the Class menu item under the Configuration menu. This opens up the class page.

Add Class

To add a class, input the details of the class in the add class form on the right side of the page and class on submit.

Once the class is added it appears under the list classes.

Edit Class

To edit a class, click on the pencil icon by the in row of the class you want to edit, to open up the edit class form. Now modify the details of the class.

Delete Class

To delete a class, click on the delete icon in the row of the class you want to delete. It bring a pop up that ask you to confirm if you really want to delete the selected class. To delete click on "OK" else click on "Cancel"

Class Search

To search for a class in the list of classes, type the name of the class in the search box, the list of classes in the class table will be filter to display the class you are searching for.

List of Class Pages

The list of class page shows ten (10) classes at a time, to see other classes us the pagination button to move through the next pages.

Copy, Print, or Export Data

1. To print the list of classes click on the print button.

2. To Export the list of classes to PDF, click on the PDF button.

3. To Export the list of classes to Excel, click on the Excel button.

4. To copy the data for the list of classes, click on the copy button.

CREDIT TYPE SETUP

Viewing Credit Types

On the left side, you'll see a list of existing credit types. Each entry shows the name, whether it's active, and options to edit or delete.

Adding a New Credit Type

On the right side, use the form to add a new credit type:

  • Enter the name of the credit.
  • Select whether it's active.
  • Click "Submit" to save or "Cancel" to discard.

Editing or Deleting

To update or remove a credit type, use the buttons next to each entry in the list.

Searching

Use the search bar above the list to quickly find a specific credit type

Department Management

This page lets you manage departments in your organization.

Viewing Departments

The list on the left shows all departments, including their active status. You can search for a department using the search bar.

Adding Department

Use the form on the right to add a new department or update an existing one. Enter the department name, choose its status, then click "Submit" to save or "Reset" to clear the form.

Removing or Editing

To delete a department, use the delete option next to its name in the list.

Education Level

This page is for managing the "Education Level" dropdown list in your system.

Adding a New Level

  1. Click the "Add Education Level" button to ensure the form is empty.
  2. Type the new name (e.g., "Masters Degree") into the "Education Level" field.
  3. Select "Active" from the dropdown menu (which currently says "Choose").
  4. Click "Submit" to save it to the database. It will appear in the list above.

Editing an Existing Level

  1. Click on an existing level in the list (e.g., click "PER-SCHOOL").
  2. The form will populate with that level's data: its name will be in the text field, and its status (Active/Inactive) will be shown in the dropdown.
  3. You can then correct a typo (e.g., change "PER-SCHOOL" to "PRE-SCHOOL") or change its status to "Inactive".
  4. Click "Submit" to save the changes.

"Cancel": Clears the form without saving any changes, cancelling either an Add or Edit operation.

Fee Code

This page is for creating and managing fee codes (e.g., "TUITION", "LIBRARY FEE").

To Add a Fee Code:

  1. Type the code name in the "Fee Code" box.
  2. Select "Active" from the dropdown.
  3. Press "Submit".

The "Search" box will filter the list once codes are added.

"Cancel" clears the form.

Expence Type

Add types of expenses (e.g., "Rent", "Supplies", "Salaries")

To Add:

  1. Enter a name in "Expense Name".
  2. Choose "Active".
  3. Click "Submit".

"Generate Report": Click to create a report of all expenses.

"Reset" clears the form.

House

View and manage a list of houses (e.g., for a school).

The List: Shows all existing houses and their colors.

"All" and "None" are likely special options for filters elsewhere.

"Generate Report": Click to create a printable or downloadable list of all houses
