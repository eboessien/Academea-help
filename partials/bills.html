Billing Module Help

About the Billing Module

The Billing module in AcademEA provides comprehensive tools for creating, managing, and applying fees to students. It allows you to set up custom fee structures, apply bills to individual students or groups, and manage billing cycles.

This help documentation covers the following specific features of the Billing module:

  • Bill Setup: Create and manage fee structures and billing items
  • Bill Students: Apply bills to multiple students at once
  • Bill Individual Student: Apply custom bills to specific students

How to Navigate to Billing Module

Log in to your AcademEA account with appropriate privileges.

From the main menu, navigate to Billing.

You will see options for Bill Setup, Bill Students, and Bill Individual Student.

Select the specific feature you want to use.

Billing Module Navigation

Bill Setup

The Bill Setup feature allows you to create and manage fee structures that can be applied to students. You can define fee codes, types, amounts, and specify which students the fees apply to.

How to Add a New Fee

Navigate to Billing → Bill Setup.

Click on "Add Fees" to create a new billing item.

Enter a unique Fee Code for identification.

Select the Fee Type from the dropdown menu.

Enter a descriptive Fee Name.

Select the applicable Academic Year and Term.

Enter the Amount for the fee.

Set the Active status to "Yes" to make the fee available for use.

Add Fees Interface - Part 1

How to Specify Fee Applicability

In the "This bill item applies to the following" section, specify which students the fee should apply to.

Select the applicable Residence type (Day, Boarding, or both).

Select the applicable Gender (Male, Female, or both).

Select the applicable Student Type (Local, International, or both).

Select the specific classes this fee applies to, or choose "Select All" to apply to all classes.

Click "Save Item" to save the fee structure or "Clear" to start over.

Add Fees Interface - Part 2

How to Find and Manage Existing Bills

Use the "Find Bill" section to search for existing fee structures.

Select the Academic Year and Term to filter bills.

Click "Apply Bill" to view bills matching your criteria.

The system will display a list of bills with details including Fee Code, Name, Amount, Class, Term, and applicability criteria.

Use the action buttons to edit or delete existing bills.

Find Bill Interface

Bill Students

The Bill Students feature allows you to apply fee structures to multiple students at once based on selected criteria.

How to Bill Multiple Students

Navigate to Billing → Bill Students.

The system will display a list of all students.

Use the search function to find specific students or browse the list.

Select the students you want to bill by checking the boxes next to their names.

Click "Bill Student" to proceed with billing the selected students.

Select the appropriate bill from the available options.

Select the Term and Academic Year for the billing.

Click "Apply Bill" to apply the selected bill to all chosen students.

Bill Students Interface

Bill Individual Student

The Bill Individual Student feature allows you to apply custom bills to specific students, which is useful for special cases or one-time charges.

How to Bill an Individual Student

Navigate to Billing → Bill Individual Student.

Select the student you want to bill from the list or use the search function.

The system will display the student's name, ID, and class.

Select the bill you want to apply from the "Select Bill" dropdown.

Select the Term and Academic Year for the billing.

Review the fee details including the fee name and amount.

Click "Apply Bill" to apply the selected bill to the student.

If immediate payment is being made, click "Make Payment" to process the payment.

Best Practices for Billing Management

Clear Fee Codes: Use descriptive and consistent fee codes for easy identification.

Regular Reviews: Regularly review and update fee structures to reflect current rates.

Specific Applicability: Be specific about which students fees apply to avoid incorrect billing.

Document Changes: Keep records of changes to fee structures for reference.

Important Notes

  • Only users with appropriate permissions can create and apply bills.
  • All billing activities are logged for audit purposes.
  • Always verify fee applicability before applying bills to students.
  • Inactive fees will not be available for application to students.

Troubleshooting Common Issues

Issue Possible Cause Solution
Cannot create or apply bills Insufficient permissions Check your user permissions or contact system administrator
Fee not appearing in dropdown Fee may be inactive or not applicable to selected students Check fee status and applicability criteria
Student not appearing in list Student may be inactive or filtered out Check student status and filters
Incorrect amount being applied Fee structure may have been updated Verify the current fee amount in Bill Setup
