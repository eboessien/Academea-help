Income & Expenditure Module Help

About the Income & Expenditure Module

The Income & Expenditure module in AcademEA provides comprehensive financial management tools for tracking all monetary transactions within your institution. It allows you to record expenses, monitor income from various sources, and generate detailed financial reports for better decision-making.

This help documentation covers the following features of the Income & Expenditure module:

  • Income & Expenditure Overview: View all financial transactions
  • Summary Reports: Categorized financial overviews
  • Expense Management: Add and view expenditures
  • Voucher Management: Print and manage vouchers
  • Other Income: Record non-fee income sources
  • Fee Analysis: Detailed breakdown of student fees

Income & Expenditure

This module provides a comprehensive view of all financial transactions, showing both income (credits) and expenses (debits) within a selected date range.

How to Use:

  1. From the side menu, select "Income & Expenditure"
  2. Set the desired date range using the Start Date and End Date fields
  3. Click the "Search" button to view transactions
  4. Use the search field to find specific transactions
  5. View detailed transaction information including dates, particulars, and amounts

Pro Tip: Export the data to Excel for further analysis by using the export function if available.

Income & Expenditure Summary

The Summary view provides a categorized overview of all income and expenses. It shows totals for different categories rather than individual transactions.

Key Features of the Summary:

Category Description
Total School fees Sum of all tuition fees collected from students
Total other income Income from non-tuition sources
Total school fees - old students Fees collected from returning students
Total feeding fee Income from meal plans and cafeteria services
Total Transport fee Income from transportation services
Total sales Income from sales of goods or services

How to Access the Summary:

  1. From Income & Expenditure, click "Summary"
  2. Or go directly from the Accounts Dashboard
  3. Set your date range and click "Search"
  4. View totals by category (fees, transport, etc.)
  5. Note the Balance, Total Income, and Total Expense figures

Pro Tip: Check this view regularly to monitor your institution's financial health and spot trends.

Add Expenditure

This form allows you to record new expenses incurred by your institution. It captures all necessary details including payment method, amount, category, and authorization information.

Step-by-Step Guide:

  1. Navigate to "Add Expenditure" from the menu
  2. Enter a voucher number for reference (if applicable)
  3. Provide a clear description of the expense
  4. Select the appropriate payment method (Cash, Cheque, Momo, etc.)
  5. Enter the expense amount in GHC
  6. Choose the relevant expense category from the dropdown
  7. Select the date of the transaction
  8. Select preview if you want to check the expenses payment
  9. Select print and pay from the preview popup to make payment

Important: Always double-check the amount and category before saving, as editing might require additional permissions.

View Expenditure

This section allows you to search, view, and manage all recorded expenses. You can filter by date range, voucher number, or other criteria.

Navigation Steps:

  1. From the Side Menu, select "View Expenditure"
  2. Set the date range for the expenses you want to review
  3. Alternatively, enter a specific voucher number in the search field
  4. Click "Search" to display matching expenses
  5. Use the "Edit" option to modify existing entries (if permitted)

Pro Tip: Use specific date ranges to narrow down results when dealing with large volumes of transactions.

Print Vouchers

This feature allows you to generate and print vouchers for financial transactions. You can search for specific vouchers by number or date range.

How to Generate Vouchers:

  1. Navigate to "Print Voucher" from the Side Menu
  2. Enter a specific voucher number OR set a date range
  3. Click "Search" to find the relevant transactions
  4. Review the results in the table format
  5. Click "Generate Report" to create a printable version
  6. Print the voucher using your browser's print function

Note: Always verify the information on the voucher before printing and filing for records.

Other Income

This form is used to record income from sources other than standard student fees, such as donations, event revenue, or other miscellaneous income.

Recording Other Income:

To add other income, follow these steps:

  1. Navigate to "Add Other Income" from the Side Menu
  2. Select the appropriate income category from the dropdown
  3. Enter the amount received in GHC
  4. Select the relevant term and academic year
  5. Specify the source of the income in "Received from"
  6. Choose the payment method used
  7. Add a clear narration describing the income source
  8. Select the transaction date
  9. Click "Save" to record the income

View Other Income

This report provides a detailed view of daily cash transactions from other income sources. It includes comprehensive information about each transaction for auditing purposes.

Accessing the Report:

  1. From the Side menu, navigate to "Daily Cash" then "Other Income"
  2. Set the desired date range for the report
  3. Click "Search" to generate the report
  4. Review the detailed information about each transaction
  5. Note the payment method details (cheque numbers, bank draft info, etc.)
  6. Identify who received each payment

Fee Breakdown

This section is designed to give you a detailed analysis of all student fee transactions. You can see how much has been collected, what types of fees have been collected, and from whom.

How to View Fee Breakdown:

  1. From the Side Menu, select "Fee Breakdown" > Income and Expenditure > under the Account option
  2. Select the Term and Academic Year for the range you want to see.
  3. (Optional) Pick a Class to see only one grade.
  4. Click "Show" to see the fee breakdown.

Pro Tip: In short, the "Fee Break Down / Summary" is your central hub for analyzing all student fee finances in detail. Always remember to use the filters at the top to specify exactly which data you want to see.

Best Practices for Financial Management

Regular Updates: Record transactions promptly to maintain accurate financial records.

Proper Categorization: Use appropriate categories for expenses to generate meaningful reports.

Regular Reviews: Check financial summaries regularly to monitor institutional health.

Backup Documentation: Keep physical copies of important vouchers and receipts.

Important Notes

  • Only users with appropriate permissions can add or modify financial records.
  • All financial activities are logged for audit purposes.
  • Always verify amounts and categories before saving transactions.
  • Use the export functions to create backups of financial data.

Troubleshooting Common Issues

Issue Possible Cause Solution
Cannot add expenses Insufficient permissions Check your user permissions or contact system administrator
Reports not generating Date range too broad or system timeout Try smaller date ranges or refresh the page
Missing transactions Incorrect date filters Verify date range settings and search criteria
Cannot edit expenses Transaction may be locked or insufficient permissions Contact administrator or check if transaction is finalized
