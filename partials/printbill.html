Print Bill Help

About Print Bill

The Print Bill feature allows you to generate and print termly bills for students. These bills include tuition fees, additional charges, payments made, and any outstanding balances.

You can generate bills for individual students, entire classes, or specific groups of students based on your selection criteria.

How to Navigate to Print Bill

Log in to your AcademEA account with appropriate administrative privileges.

From the main menu on the left side of the page, locate and click on the Accounts section.

Within the Accounts menu, find and select Print Bill.

You will be directed to the Print Bill page where you can select criteria and generate bills.

Navigation Path Preview

How to Generate and Print Bills

Once you've navigated to the Print Bill page, follow these steps to generate bills:

You will see a form with selection criteria options as shown in the preview below.

Select the appropriate Class from the dropdown menu.

Choose the relevant Term from the term dropdown.

Select the Academic Year from the year dropdown.

Click the Print Bill button to generate and print the bills.

Form interface for selecting bill criteria

Advanced Options

The Print Bill feature includes several advanced options for customizing your bills:

Option Description Use Case
Individual Student Bills Generate a bill for a specific student rather than an entire class When parents request a copy of their child's bill
Custom Charges Add or edit additional charges before printing When you need to include special fees or adjustments
Multiple Format Output Save bills as PDF, Excel, or print directly When you need digital copies for records or emailing
Batch Printing Print bills for an entire class or grade level at once Beginning-of-term billing for all students
Bill Templates Choose from different bill layout templates When you want to customize the look of bills

Tips for Effective Bill Printing

Preview before printing: Always check the bill preview to ensure all information is correct and properly formatted.

Use high-quality paper: For official bills, use your institution's letterhead or high-quality paper.

Check printer settings: Ensure your printer has enough ink and paper before starting large print jobs.

Save digital copies: Always save PDF copies of bills for your records before printing.

Verify data accuracy: Double-check that all fees, payments, and balances are up-to-date.

Important Notes

  • Bills can only be generated after fee structures have been set up for the selected term.
  • Some billing features may require administrative privileges to access.
  • Custom bill templates may need to be configured in the System Setup section.
  • Always verify the selected term and academic year before printing to avoid errors.
  • Ensure all payments have been properly recorded before generating bills.

Troubleshooting Common Issues

Reprint Receipt Help

About Reprint Receipt

The Reprint Receipt feature allows you to search for and reprint fee receipts that have already been issued. This is useful when a receipt has been lost, damaged, or needs to be duplicated for record-keeping purposes.

You can search for receipts by receipt number, student details, or date range, and reprint them with the original information intact.

How to Navigate to Reprint Receipt

Log in to your AcademEA account with appropriate administrative privileges.

From the main menu, navigate to Staff > Reports.

From the Reports submenu, select Reprint Fee Receipt.

You will be directed to the Reprint Receipt page where you can search for and reprint receipts.

Navigation Path Preview

How to Search and Reprint Receipts

Once you've navigated to the Reprint Receipt page, follow these steps:

Enter the Receipt Number in the search field if you know it.

Alternatively, use the Filter by options to search by other criteria such as student name, date, or class.

Click the Search button to find matching receipts.

Review the search results in the table below.

Click the Action button next to the receipt you want to reprint.

Select Print or Reprint from the action menu.

Reprint Receipt search interface

Advanced Search Options

The Reprint Receipt feature includes several search and filter options:

Option Description Use Case
Receipt Number Search Search by specific receipt number When you know the exact receipt number
Date Range Filter Filter receipts by date range When searching for receipts from a specific period
Student ID/Name Search Search by student identifier When you need all receipts for a specific student
Class Filter Filter by student class When you need receipts for an entire class
Payment Method Filter Filter by Momo, Cheque, or Cash When reconciling specific payment methods

Tips for Reprinting Receipts

Verify receipt details: Always check that all information on the receipt is correct before reprinting.

Use original receipt paper: For official reprints, use the same security paper as original receipts.

Mark as duplicate: Consider stamping "DUPLICATE" on reprinted receipts to avoid confusion with originals.

Keep records: Maintain a log of all reprinted receipts for auditing purposes.

Secure access: Limit access to receipt reprinting to authorized staff only.

Important Notes

  • Reprinted receipts contain the same information as the original receipt.
  • Some institutions may require authorization before reprinting receipts.
  • Reprinting a receipt does not create a new transaction; it only reproduces existing transaction details.
  • Always verify the identity of the person requesting a receipt reprint.
  • Keep track of all reprinted receipts for audit trail purposes.

Troubleshooting Common Issues

No search results No receipts match the search criteria Broaden search criteria or check date range
Cannot reprint receipt Insufficient permissions or technical issue Contact system administrator or IT support
Incorrect information on receipt Original transaction was recorded incorrectly Use the Recall Fees feature to correct the transaction first

Recall Fees Help

About Recall Fees

The Recall Fees feature allows you to review, adjust, and reverse fee transactions that have been recorded in the system. This is essential for correcting errors, processing refunds, or adjusting payments that were recorded incorrectly.

With this feature, you can search for specific transactions, view their details, and make necessary adjustments while maintaining a complete audit trail of all changes.

How to Navigate to Recall Fees

Log in to your AcademEA account with appropriate administrative privileges.

From the main menu, navigate to Staff > Reports.

From the Reports submenu, select Recall Fees.

You will be directed to the Recall Fees page where you can search for and adjust fee transactions.

Navigation Path Preview

How to Search and Adjust Fee Transactions

Once you've navigated to the Recall Fees page, follow these steps:

Enter the Receipt Number in the search field if you know it.

Alternatively, use the available filters to search by other criteria such as date, student name, or class.

Click the Search button to find matching transactions.

Review the search results in the table below.

Click the Action button next to the transaction you want to adjust.

Select the appropriate action (Adjust, Reverse, or View Details).

Make the necessary changes and provide a reason for the adjustment.

Confirm the changes to update the transaction.

Recall Fees search and adjustment interface

Adjustment Options

The Recall Fees feature offers several adjustment options:

Option Description Use Case
Partial Adjustment Modify part of a transaction amount When only a portion of the payment needs correction
Full Reversal Completely reverse a transaction When a payment was recorded in error or needs refund
Payment Method Change Change how payment was made (Momo, Cheque, Cash) When payment method was recorded incorrectly
Date Correction Adjust the transaction date When transaction was recorded with wrong date
Student Reassignment Reassign payment to different student When payment was applied to wrong student account

Best Practices for Fee Adjustments

Document reasons: Always provide a clear reason for any fee adjustment for audit purposes.

Verify original transaction: Double-check the original transaction details before making adjustments.

Get authorization: Ensure you have proper authorization for fee adjustments based on your institution's policies.

Notify relevant parties: Inform affected students or parents about adjustments when necessary.

Review impact: Check how adjustments affect student balances and financial reports.

Important Notes

  • Fee adjustments may affect financial reports and student balances.
  • All adjustments are logged with user ID, timestamp, and reason for audit purposes.
  • Some adjustments may require additional authorization based on amount or type.
  • Reversing transactions may affect related records like receipts and invoices.
  • Always verify the impact of adjustments on financial statements.

Troubleshooting Common Issues

Issue Possible Cause Solution

Unexpected balance changes Adjustment affected related transactions Review transaction history and make additional adjustments if needed
