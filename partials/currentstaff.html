"Current Staff" Page

This page serves as the main directory for viewing and managing all existing staff members.

Purpose:

To provide a searchable list of all current staff, allowing users to quickly find and select an individual to view or modify their full details.

Key Sections:

Overview: The header provides a user greeting and a clear description of the page's function: "View & modify staff details here."
Search Function: A Search bar is available to filter the list of staff by typing in a name, ID, or other criteria.
Staff Table: The core content is a table listing staff members with their basic information: ID, NAME, and EMAIL/USERNAME.
Pagination: The "Showing 1 to 10 of 99 entries" text and the Previous, Next, and page number buttons indicate a long list that is split across multiple pages for easy navigation.

Functionality:

This is a central hub for staff management. A user would likely click on a specific staff member's name or row in this list to access and edit their complete profile information. The search and pagination features make it efficient to navigate a large number of employee records.
