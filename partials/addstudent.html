Add Student Guide

Learn how to add new students to the AcademEA system

Time: 10 min read Difficulty: Easy Last updated: Aug 26, 2025

Introduction to Adding Students

The Add Student feature allows you to register new students into the AcademEA system. This process collects all necessary information to create a complete student profile.

Note: You need appropriate permissions to add students. Contact your system administrator if you don't see the Add Student option.

In this guide, you'll learn how to:

  • Navigate to the Add Student section
  • Complete the multi-step form
  • Enter student information
  • Add parent/guardian details
  • Save and complete the process

Navigating to Add Student

To access the Add Student section:

1Open the Dashboard

Log in to your AcademEA account and you'll be directed to the Dashboard homepage.

2Access the Students Menu

In the sidebar navigation, locate and click on the "Students" menu to expand it.

3Select Add Student

From the expanded Students menu, click on "Add Student" to open the form.

Navigation menu

Form Steps Overview

The Add Student form is divided into multiple steps to make the process organized and manageable:

1Student Information

This is the first step where you enter the student's personal details:

  • First Name, Last Name
  • Gender, Date of Birth
  • Phone Number
  • Class/Form selection
2Father Information

The second step collects information about the student's father:

  • Father's Name
  • Contact Information
  • Occupation and Education
3Mother Information

The third step collects information about the student's mother:

  • Mother's Name
  • Contact Information
  • Occupation and Education
4Guardian Information

The final step collects information about the student's guardian (if different from parents):

  • Guardian's Name
  • Contact Information
  • Relationship to student

Form steps visualization

Student Information

The first step requires you to enter the student's personal information. Here's what you need to know:

Student Information Form Preview
Tip: Fields marked with an asterisk (*) are required. You must complete these fields to proceed to the next step.
1Complete Required Fields

Fill in all fields marked with an asterisk. These are essential for creating the student record.

2Provide Additional Information

While optional fields are not required, providing complete information helps create a more comprehensive student profile.

3Navigate to Next Step

Click the "Next" button to proceed to the Father Information step once all required fields are completed.

Parent Information

The next steps involve entering information about the student's parents or guardians:

1Father Information

Complete information about the student's father including:

  • Full name and title
  • Mobile number
  • Postal and residential addresses
  • Education level and occupation
2Mother Information

Complete information about the student's mother including:

  • Full name and title
  • Mobile number
  • Postal and residential addresses
  • Education level and occupation
3Guardian Information

If the student's guardian is different from their parents, provide:

  • Guardian's full name and title
  • Mobile number
  • Relationship to the student
  • Postal and residential addresses
Note: You can use the "Previous" button to go back to earlier steps and make changes if needed.

Parent information form example

Completing the Process

After completing all form steps, you can finalize the student registration:

1Review Information

Use the "Previous" button to navigate through all steps and verify that all information is correct.

2Save Student Record

Click the "Save Student" button to complete the registration process.

3Confirmation

You should receive a confirmation message indicating the student has been successfully added to the system.

Tip: You can save a partially completed form as a draft and return to it later by clicking the "Save Draft" button.
Important: After saving, you can add additional information like medical details from the Current Students page.

Troubleshooting

Here are solutions to common issues you might encounter:

Ensure all required fields (marked with *) are completed. Check for any validation messages highlighting missing or incorrect information.

Check your internet connection. If the problem persists, try refreshing the page and ensure all required fields are completed before saving.

Try refreshing the page. If the issue continues, check if you have any browser extensions that might be interfering with dropdown functionality.
