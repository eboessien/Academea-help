Add Student

To add a student click on the add student menu below



It opens the add student form



This form has four (4) steps

  1. Step 1: Add Student Information
  2. Step 2: Add Father Information
  3. Step 3: Add Mother Information
  4. Step 4: Add Guardian Information.

To add other information like medical information go to the Current Student Page to add it
