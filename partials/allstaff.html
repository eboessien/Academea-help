All Staff Documentation

Overview

The All Staff page displays a comprehensive list of all current staff members in the system. In addition to viewing staff information, various management tasks can be performed from this interface.

Figure 1: All Staff page interface

Management Tasks

The All Staff page provides several management functions to help you work with staff data efficiently:

Print Staff List

Generate a printer-friendly version of the staff list for physical records or distribution.

Copy Data

Copy the staff list data to clipboard for pasting into other applications or documents.

Export to Excel

Export the staff list to Microsoft Excel format for further analysis or reporting.

Export to PDF

Create a PDF document of the staff list for official records or sharing.

Move to Archive

Transfer selected staff members to the old staff database when they are no longer active.

Export Options

The All Staff page offers multiple export formats to suit different needs:

Print: Creates a printer-friendly version of the staff list with optimized formatting for paper.

Copy: Copies the visible data to your clipboard for quick pasting into other applications.

Excel Export: Generates a complete Excel spreadsheet with all staff data for further analysis.

PDF Export: Creates a professional PDF document suitable for official records or sharing.

Tip

When exporting data, only the currently visible columns will be included in the export. Use the column visibility options to customize what data is exported.

Column Visibility Control

You can customize which columns are visible on the All Staff page to focus on the information that matters most to you.

Figure 2: Column visibility controls

How to Show or Hide Columns

Locate the column visibility controls (usually represented by checkboxes or a settings icon).

Check the checkbox next to a column name to make it visible.

Uncheck the checkbox to hide that column from view.

Your column preferences are typically saved for your next visit.

Important Note

Only visible columns will be included when you print, copy, or export the staff data. Make sure to show all necessary columns before performing these operations.

Example Use Case

If you only want to see staff members' first names, last names, and departments:

Open the column visibility controls.

Uncheck all columns except "First Name", "Last Name", and "Department".

Apply the changes to update the staff list view.
