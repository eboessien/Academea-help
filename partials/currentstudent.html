Introduction to Current Students

The Current Students section allows you to view and manage all students currently enrolled in your institution. This powerful tool helps you access student information quickly and efficiently.

Note: You must have appropriate permissions to view and edit student information. Contact your system administrator if you need access.

In this guide, you'll learn how to:

  • Select a class to view its students
  • Search for specific students
  • View detailed student information
  • Add student photos
  • Navigate between different information tabs

Viewing Students by Class

To view students in a specific class, follow these steps:

1Select a Class

Use the "Choose Class" dropdown menu to select the class you want to view. The student list will automatically update to show only students from that class.

Class selection dropdown example

2Search for a Student

If you're looking for a specific student, use the Search box above the student list. You can search by:

  • Student ID
  • First or Last name
  • Other identifying information

Search box example

3View Student List

Once you've selected a class, the student list will display all students in that class. The list shows:

  • Student ID
  • Student Name
  • Class Information

Viewing Student Details

To view detailed information about a student:

1Select a Student

Click on a student's name in the student list. Their details will appear on the right side of the screen (or below on mobile devices).

2Review Student Information

The student details section shows important information such as:

  • First and Last name
  • Student ID
  • Class/Form
  • Gender
  • Residence Type
  • Phone Number
  • Program Type
Tip: Required fields are marked with an asterisk (*). You must complete these fields when adding or editing student information.

Adding Student Photos

You can add photos to student profiles in two ways:

1Upload from Your Device

To upload a photo from your computer or mobile device:

  1. Scroll to the bottom of the student details section
  2. Click the "Choose File" button
  3. Select the image file from your device
  4. Click "Upload" to save the photo
2Take a Photo with Your Camera

To take a new photo using your device's camera:

  1. Click the "Start Camera" button
  2. Allow the browser to access your camera when prompted
  3. Position the student in the frame
  4. Click "Take Photo" to capture the image
  5. Click "Stop Camera" when finished
Important: Make sure you have permission to take and store student photos according to your institution's privacy policy.

Information Tabs

Each student's profile contains multiple tabs with different types of information:

Student Tab

Contains basic student information like name, ID, class, and contact details.

Father Tab

Contains information about the student's father, including contact details.

Mother Tab

Contains information about the student's mother, including contact details.

Guardian Tab

Contains information about the student's guardian if different from parents.

1Switch Between Tabs

Click on any tab to view the corresponding information. The active tab will be highlighted.

Troubleshooting

Here are solutions to common issues you might encounter:

Make sure you've selected a class from the dropdown menu. If the problem persists, contact your system administrator to verify that students are properly enrolled in the selected class.

Ensure that you've allowed browser access to your camera. Check your browser settings and privacy permissions. If using a work computer, your organization might restrict camera access.

You may not have editing permissions. Contact your system administrator to verify your user role has appropriate permissions to edit student information.
